## インライン要素とブロック要素
> 2026-04-03

HTMLの要素には2種類ある。

- **ブロック要素**（div, p, h1など）— 横幅いっぱいに広がる。`margin: auto` で中央寄せできる
- **インライン要素**（span, a, audioなど）— 文字と同じ扱い。`margin: auto` が効かない

**中央寄せしたいときは子ではなく親をいじる**
- 親がflex → `justify-content: center`
- 親がblock → `text-align: center`

また、`<p>`タグを閉じ忘れると後続の要素が`<p>`の中に入り込み、flexの直接の子にならないので注意。

## `<ul><li>` の正しい使い方
> 2026-04-03

**Q**: `ul` や `li` を使ったらだめなの？

**A**: ダメではない。ただし構造が正しい必要がある。

```html
<!-- NG: ulの外にliが単独でいる（無効なHTML） -->
<li>
  <ul>
    <li>内容</li>
  </ul>
</li>

<!-- OK: ulの中にliが入る -->
<ul>
  <li>内容</li>
  <li>内容</li>
</ul>
```

画像を並べるだけなら `div` のほうがシンプル。「作品リスト」として意味を持たせたいなら `ul/li` でも良い。

## `<style>` タグはテキスト表示に使えない
> 2026-04-03

**Q**: `<style font: size 5px;>テキスト</style>` と書いたらエラーになった

**A**: `<style>` はCSSを書く場所であり、テキストを表示するものではない。属性の書き方も無効。
小さい文字を表示したい場合は `<small>` か `<span style="font-size: 0.8em;">` を使う。
